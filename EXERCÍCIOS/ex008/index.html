Exercício sobre as principais formatações do HTML5

Negrito/Destaque

Primeiramente, vamos fazer de uma forma não semântica que é usando a tag <b>

aqui temos um termo em negrito no entanto, ele foi escrito de forma não semântica

já aqui, usando a tag <strong> , temos um elemento em negrito, de forma semântica, pois foi utilizado um significado pra isso, e nao somente a forma

veja só Esse termo está entre a tag strong, por isso está em negrito

Agora, veremos o itálico/ênfase

aqui, veremos um termo em itálico, porém que foi colocado de maneira não semântica, utilizando a tag <i>
veja: eu quero caféeee eeeeeeee

porém, para utilizar esse recuso de maneira que tenha sentido semântico, basta usar a tag <em>
assim: termo em negrito semânticamente

Texto marcado/marca texto

também é possível criar textos marcados, utilizando a tag <mark>
assim: esse texto foi marcado

Tamanho de texto

estamos criando um texto grande e um texto pequeno

Texto excluido

isso serve para que a gente indique que um texto deve ser lido, porém não deve ser considerado
para isso, colocamos o texto entre a tag <del>
Dessa forma: esse texto aparecerá riscado na tela

Texto inserido

esse tipo de texto aparecerá sublinhado no seu site, e para isso ele fica dentro da tag <ins>
Dessa forma: isso aqui é importante parcero

Texto sobrescrito

funciona para colocar expoente em um número, por exemplo: X2

Texto subescrito

aqui é parecido, no entanto o número fica embaixo, por exemplo: O3
